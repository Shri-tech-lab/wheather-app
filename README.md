# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript.  
This application allows users to search for any city and view real-time weather information.

Live Link: [https://shri-tech-lab.github.io/wheather-app/](https://shri-tech-lab.github.io/wheather-app/)

## 📌 Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature
- ☁️ Weather condition display
- 💨 Humidity and wind speed
- 📱 Responsive design
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/shri-tech-lab/wheather-app.git
```

2. Open project folder

```bash
cd wheather-app
```

3. Open `index.html` in browser
